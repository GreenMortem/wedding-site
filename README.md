# Свадебный сайт (шаблон)

Это адаптивный одностраничный свадебный сайт‑приглашение в стиле референса (бежевый «лист», зелёные плашки) + анкета RSVP на сайте с сохранением ответов на сервере.

## Что поменять в первую очередь

1) Откройте `script.js` и заполните блок `CONFIG`:
   - `names` — имена
   - `dateText` — дата (текстом)
   - `places` — адреса/ссылки на карты (2 локации: регистрация и банкет)
   - `timeline` — тайминг (можно добавлять пункты)
   - `contacts` — контакты
   - `rsvpDeadline` — до какой даты собрать ответы

2) Откройте `index.html` и поправьте тексты секций:
   - приглашение
   - место/адрес (если нужно добавить пояснения)
   - тайминг (контент формируется из `script.js`)
   - дресс‑код (палитра настраивается в `script.js`)
   - подарки/пожелания
   - контакты

3) Фотографии:
   - Сейчас стоят «заглушки». Можно заменить их на ваши фото — в `index.html` найдите блоки с классом `.photo`.

## Дресс‑код (кружки «как на примере»)

Сайт ожидает файл с текстурами кружков:
`wedding-site-template/assets/dresscode.jpg`

Сохраните картинку с кружками (ту, что вы прислали) в эту папку под именем `dresscode.jpg`.

## Как запустить

Сайт рассчитан на запуск через Node.js сервер (он раздаёт фронт и принимает ответы анкеты).

1) Установите Node.js
2) В папке проекта выполните:
   - `npm install` (если в Windows блокируются ps1-скрипты, используйте `npm.cmd install`)
3) Создайте `.env` по примеру `.env.example` и задайте `ADMIN_CODE`
4) Запуск:
   - `npm run dev` (или `npm.cmd run dev`)
5) Откройте:
   - `http://localhost:3000/` — приглашение
   - `http://localhost:3000/admin` — админка ответов (вводите `ADMIN_CODE`)

## Публикация

Так как ответы сохраняются на сервере, проект нужно размещать на вашем сервере с Node.js.
